# docs/demo.gif — ручная запись

`docs/demo.gif` **записывается вручную перед подачей на конкурс**; он не производится безголовым
CI-шагом. Гифка должна наглядно показывать работу killer-флоу без присмотра — LiveBadge канваса
показывает **«24/7 · N аккаунтов»**, пока вкладка браузера закрыта, а бамп лота приходит по
расписанию. Этот момент «работает без твоего ПК» — вся суть доказательства, поэтому она
записывается с настоящего работающего канваса живым оператором.

Запустите `scripts/record_demo_gif.sh` для пошаговой процедуры записи. Цель: **≤15 с**, ~800×500,
≤8 МБ (оптимизируйте `gifsicle -O3`, если нужно). Закоммитьте результат в `docs/demo.gif`; на него
ссылается [README.md](../README.md).

Пока гифка не записана, ссылка на картинку в README ведёт на отсутствующий файл — это ожидаемо во
время разработки и закрывается в чек-листе перед подачей.
